# Connecting to the Robot

  1. Turn on the robot by following the instructions here.
  2. Turn on your tablet and wait until you reach the Windows desktop.
  3. Click on the WiFi icon in the bottom right and connect to robocon-TEAMXX where XX is your team number.
  4. Navigate to http://robot.go in your web browser of choice (Chrome or Firefox are recommended). You should see an interface similar to this:

From now on, this will be referred to as the Shepherd Homepage.

  • If you want to upload and run existing code, follow the Uploading and Running Code tutorial
  • If you want to write Python code or use the Scratch-like interface, follow the Using the Editor tutorial
